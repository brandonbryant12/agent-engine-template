# Engineering Standards Docs

This directory is the standards source of record for repositories created from the chat template.

Template scope:
- authenticated chat experience
- streaming assistant responses
- background run queue + worker
- typed API contracts and effect-based handlers

## Directory Layout

- `docs/architecture/` - boundaries, security, telemetry
- `docs/patterns/` - backend implementation patterns
- `docs/frontend/` - web app architecture and conventions
- `docs/testing/` - testing expectations by change type
- `docs/master-spec.md` - product behavior contract
